xcode.swift
xcode.swift is an open source organization that aims to build libraries and tools in Swift to make working with Xcode easier and a more pleasant experience. All the tools are built in the open and support multiple dependency management tools like Swift Package Manager, CocoaPods or Carthage.

Check out other tools on the GitHub organization and stay up to date following the twitter account @xcodeswiftio. You are also welcome to join the organization Slack where you can talk to the contributors and other users of the tools.
